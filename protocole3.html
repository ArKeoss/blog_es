Projet sur les Variations Climatiques

Sara BEN MEKKI; Sofia AJDID; Fadwa CHATWITI; Wiam BELMEKKI

OBJECTIF: Expliquer les variations climatiques aux élèves de 5e.

THÈME: Les variations climatiques

Expérience 1: le nuage dans un bocal

Matériel:

  • Bocal ou récipient
  • Eau chaude
  • Allumette
  • Assiette ou surface froide pour couvrir le récipient

Comment faire?

  1. Remplissez le récipient d’eau chaude puis mélangez
  2. Allumez l’allumette et soufflez-la puis faites-la tomber directement dans l’eau
  3. Couvrez rapidement le bocal avec l’assiette froide
  4. Regardez le nuage de vapeur d’eau se former dans le bocal

Qu’est-ce qui se passe?

La vapeur d’eau va se mélanger aux particules de fumée. Cette vapeur va atteindre la surface froide, va se refroidir et se condenser. Il y a alors formation d’un nuage. C’est exactement ce qui se passe sur Terre mais dans ce cas la vapeur d’eau se mélange à d’autres particules (poussières, pollution...)

Expérience 2: L’effet de serre

Matériel:

  • Deux bocaux
  • Deux thermomètres
  • Papier aluminium
  • Source de lumière

Comment faire?

  1. Placez les deux thermomètres dans les bocaux
  2. Couvrez un des deux bocaux de papier aluminium
  3. Laissez les bocaux sous la source de lumière
  4. Vérifiez l’évolution de la température

Qu’est-ce qui se passe?

Le bocal couvert imite ce qui se produit avec l’effet de serre. L’air est réchauffé par la source de lumière, mais contrairement au bocal non couvert, l’air chaud ne peut s'échapper, il continue donc à se réchauffer.

Expérience 3: fonte des glaces

Matériel:

  • Récipient transparent
  • Objet à utiliser comme “Terre”
  • De l’eau
  • Des glaçons

Comment faire?

  1. Placez l’objet “Terre” dans le récipient et remplissez ce dernier d’eau
  2. Placez ensuite les glaçons sur l’objet représentant la Terre. Notez le niveau d’eau dans le récipient
  3. Attendez simplement que la glace fonde et observez comment le niveau d’eau change

Qu’est-ce qui se passe?

Les glaçons qui fondent aident à représenter la fonte des glaces terrestres dans l’Arctique.
